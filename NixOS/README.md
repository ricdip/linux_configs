# NixOS configuration

My NixOS configuration using Nix flakes

## Help message (just)

```
Available recipes:
    default             # Show help message
    history             # List system generations
    rebuild             # Rebuild system
    rebuild-flake FLAKE # Rebuild system specified by FLAKE
```
